常用快捷键

2019-03-19 (Updated: 2021-10-28)

windows

ctrl + shift + esc 进程管理

win + D 全部窗口最小化/还原

alt + esc 窗口切换

win + F 搜索

Win + Home最小化所有窗口，除了当前激活窗口。

Win + ↑最大化窗口。

Win + ↓最小化窗口

win + ← 左靠边

win + → 右靠边

win + shift + ↑ 上下拉伸窗口到最大

win + space 窗口透明化

**Ctrl + Shift + T **在浏览器中快速恢复刚刚关闭的一个网页

Ctrl + W 关闭标签页

F2 选中文件重命名

Chrome浏览器

**Alt+L ** 光标定位到地址栏Location

Ctrl+N 打开新窗口

Ctrl+T 打开新标签页

Ctrl+1 ~ Ctrl+8 切换到指定位置编号的标签页，按下的数字代表标签页横条上的标签位置。

ctrl+shift+O 打开书签管理页面

alt + ← 后退

phpstorm

ctrl +h 显示类层级关系

end +shift home 或 **home +shift end **选中一行

Ctrl +ｗ 选中代码，通过多击w键选择不同范围

Ctrl +　shift + ｗ减少选中代码，通过多击w键选择不同范围

Shift +F2 　显示跳至带代码书写错误的地方

Alt + Shift + Down line 向下移动

**Alt + Shift + Up line **向上移动

ctrl alt l 格式化代码

ctrl alt <- 后退

ctrl + shift + / 注释

ctrl + .? 折叠或展开选中的代码为…

ctrl + shift + +,- 展开或缩起(){}

Ctrl + Shift + n 查找文件

Ctrl + j 插入活动代码提示

**Ctrl + Alt + t ** 当前位置插入环绕代码

Alt + Insert 生成代码菜单

**Ctrl + q **查看代码注释

**Ctrl + d **复制当前行

Ctrl + y 删除当前行

Shift + F6 批量重命名

Ctrl + Shift + u 字母大小写转换

Ctrl + f 查找

**Ctrl + r **替换

**Ctrl + Shift + i **查看变量或方法定义源

Ctrl + g 跳转行

Ctrl + Alt + F12 跳转至当前文件在磁盘上的位置

Alt + Down 查看下一个方法

Alt + Up 查看上一个方法

Ctrl + / 行注释

Ctrl + Shift + / 块注释

**Ctrl + b **跳到变量申明处

Ctrl + [] 匹配 {}[]

Ctrl + Shift + ]/[ 选中块代码

**Ctrl + Shift + v **复制多个文本

**Alt + left/right **标签切换

Ctrl + p 显示默认参数

Ctrl + F12 在当前类文件里快速查找方法
